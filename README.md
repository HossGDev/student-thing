# Python Learning Tool - From Basics to Big O!

Interactive CLI-based learning tool for Python, covering everything from basic syntax to Big O complexity.

## Features

- **18+ Educational Scenarios**: Exercises ranging from list sorting and dictionaries to recursion and Big O analysis.
- **Interactive Quizzes**: Multiple-choice and code-writing challenges for each topic.
- **Instant Feedback**: Get immediate validation on your answers with sample solutions provided.
- **Big O deep dive**: Learn how to identify and compare O(1), O(n), and O(n²) time complexities.

## Installation & Running

The fastest way to run this tool is using [`uv`](https://github.com/astral-sh/uv).

### 1. Get the Code
You can either clone the repository or download the ZIP file from GitHub.

**Option A: Clone the Repo**
```bash
git clone https://github.com/HossGDev/student-thing.git
cd student-thing
```

**Option B: Download ZIP**
1. Go to [https://github.com/HossGDev/student-thing](https://github.com/HossGDev/student-thing)
2. Click **Code** > **Download ZIP**
3. Extract the files and open a terminal in the folder.

### 2. Run the Tool
Once you have the code, run it using `uv`:

```bash
uv run main.py
```

*If you don't have `uv` installed, you can get it with:*
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
